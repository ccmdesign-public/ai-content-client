## Summary

This week brought a massive wave of AI news, highlighted by major leaks, new model releases, and significant platform updates across the industry.

### Anthropic and OpenAI Leaks

Anthropic experienced a massive leak revealing a 10-trillion parameter model that exposed two upcoming tiers: **Claude Mythos** and **Capabara**. **Mythos** represents a completely new frontier tier with massive enhancements in coding, academic reasoning, and cyber security, prompting Anthropic to plan a very slow rollout due to security risks. Meanwhile, **Capabara** sits just below **Mythos** but remains a full class above the current flagship Opus models. On the other side, OpenAI is reportedly developing a new internal model named **Spud**, and is entirely shutting down its **Sora** video generation app to free up compute resources for this development.

### Open Source and Real-Time Models

The open-source community saw major releases this week with the launch of the **GLM 5.1** model, a highly capable agent designed for multi-step workflows. It scored a 45.3 on coding benchmarks, rapidly closing the gap with proprietary models like Opus 4.6. Mistral AI also contributed to the open-source ecosystem by releasing **boxrol TTS**, an ultra-fast, expressive text-to-speech model supporting nine languages. In the proprietary space, Google DeepMind introduced **Gemini 3.1 Flash Live**, a real-time multimodal model specifically optimized for low-latency voice and vision agents.

### Coding Platform Updates

Developer tools received massive upgrades to become far more autonomous. OpenAI is shifting **Codex** from an isolated coding tool into a full execution environment by introducing plugins, allowing developers to launch entire pre-built workflows directly inside the app. Anthropic responded with heavy updates to **Claude Code**, including cloud-based automated pull request fixing and a new auto mode that safely executes code without constant permission prompts. Additionally, Anthropic is developing **Operon**, a specialized desktop agent designed specifically for biological scientific research. Finally, the popular coding editor Cursor faced controversy after users discovered their new Composer 2 model was actually a secretly fine-tuned version of the open-source **Kimik K 2.5**.

### The ARC AGI 3 Benchmark

The release of the **ARC AGI 3** benchmark drastically changed how the industry measures AI intelligence. Unlike previous tests, it drops models into interactive environments with no prior training to test true agentic reasoning and prevent fake intelligence via pattern overfitting. While humans achieve a 100% solve rate on the first try, current top AI models score under 1%, proving that current systems still heavily rely on memorization rather than adaptability.

## Context

The AI industry is rapidly transitioning from simple chatbots to autonomous, agentic systems capable of executing complex workflows. The emergence of massive new models like Claude Mythos suggests we are approaching a new frontier of intelligence that could further automate white-collar labor. However, the dismal AI performance on the new ARC AGI 3 benchmark highlights a critical gap between pattern memorization and true interactive reasoning. Developers and businesses must pay close attention to this shifting ecosystem, as platforms like Codex and Claude Code evolve into full execution environments, and open-source models like GLM 5.1 become viable, cost-effective alternatives to proprietary giants.
