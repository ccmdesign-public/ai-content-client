## Key Takeaways

Anthropic's Claude Code received significant feature updates that enhance its capabilities as an AI coding assistant and productivity tool. • **Remote Control** allows accessing Claude Code sessions from phones or web apps via `/remote` command • **Auto Memory** enables Claude to remember project details across sessions in a markdown file • **New Plugins** include Knowledge Work and Financial Services plugins with customizable natural language skills • **Upcoming Skills** Simplify and Batch will use parallel agents for code quality and migration tasks

## Summary

Anthropic has rolled out a substantial update to Claude Code with features that expand its functionality beyond traditional coding assistance.

**Remote Control** enables users to connect to their computer's Claude Code session from mobile devices or web applications by running the `/remote` command. This mirrors functionality previously seen in OpenClaw and provides flexibility for developers on the go.

**Scheduled Tasks** within Co-Work allow for automated workflows like research summaries, file organization, and email follow-ups at specified intervals. This brings task automation capabilities similar to what was available in ChatGPT's task features directly into the Claude ecosystem.

**New Plugin Repositories** include Knowledge Work Plugins and Financial Services Plugins that can be used in both Claude Code and Co-Work. These plugins offer industry-specific skills that users can install from the marketplace, customize through natural language editing, and fork for their own needs. The equity research plugin, for example, includes idea generation workflows that can be easily modified.

**Auto Memory Feature** allows Claude to remember project-specific information across different sessions, including debugging patterns, context details, and preferred approaches. This operates separately from claw.md or skills files and creates a dedicated markdown file that Claude can reference. Users can prompt Claude to remember specific items, and the system will automatically learn helpful patterns over time while remaining editable.

**UI Enhancements** include updates to the ask user question tool that now displays markdown snippets for diagrams and code examples. This provides visual representations when discussing layout changes or front-end components, offering two-dimensional suggestions for presentations, websites, and other projects.

**Adoption Statistics** reveal that 4% of GitHub public commits are now authored by Claude Code, with projections suggesting this could reach 20% by the end of 2026. This rapid adoption indicates how significantly AI is transforming software development practices.

**Future Skills** announced by creator Boris Jurnney include Simplify and Batch commands. The Simplify skill will use parallel agents to improve code quality, tune efficiency, and ensure claw.md compliance. The Batch skill enables interactive planning of code migrations with parallel execution using dozens of isolated agents that run in git work trees, test workflows, and create pull requests.

## Context

These updates position Claude Code as a comprehensive AI development environment rather than just a coding assistant. The features address key developer pain points like context switching between devices, maintaining project continuity, and automating repetitive tasks. This matters because AI-assisted development is accelerating rapidly, with Claude Code already responsible for 4% of GitHub commits and projected to reach 20% by end of 2026. Developers, technical managers, and anyone involved in software creation should understand these capabilities as they represent the future of programming workflows.
