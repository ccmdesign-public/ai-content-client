## Key Takeaways

The video demonstrates a new interactive feature in Claude Code. • The **/btw command** lets you ask questions while a main prompt is executing. • It provides **immediate answers without interrupting** the primary workflow. • It's designed for **quick, contextual questions** about your project, like checking a Laravel version.

## Summary

This quick demo showcases a new feature in Claude Code called the **/btw command**. The core function allows users to ask questions 'by the way' while a longer, main prompt is actively running in the background.

When a user starts a major coding task, they can use `/btw` to ask quick, contextual questions related to their project. For example, the video shows typing `/btw what is Laravel version` to get an immediate answer. This answer appears in a side panel or chat stream without pausing or interrupting the primary task Claude Code is performing.

The presenter notes this is a new feature that often appears as a tip within the Claude Code interface. It's highlighted as a small but useful addition for maintaining workflow continuity, allowing developers to clarify details or gather information on the fly.

Additional context from the original tweet by Tariq is mentioned, noting that the feature is currently limited: it's mostly for quick questions and cannot call external tools. It's also described as a side project by Eric from the Claude Code team. The video concludes by prompting viewers to discuss potential use cases for this type of in-workflow questioning in the comments.

## Context

This matters because it addresses a common friction point in AI-assisted coding: the need to interrupt a long-running task to ask a clarifying question. For developers using Claude Code for complex prompts, this feature streamlines the workflow, allowing for parallel inquiry and execution. It represents a step toward more conversational and context-aware AI coding assistants, where the AI can handle multitasking within a single session.
