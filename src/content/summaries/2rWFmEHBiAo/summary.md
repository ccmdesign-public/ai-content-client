## Key Takeaways

This video details Ahrefs's September 2025 product updates, focusing on new capabilities for SEO and marketing professionals. Key highlights include:

*   **Brand Radar 2.0 enhancements** like **Entities** (grouping brand name variants) and **Topics** (clustering queries for content opportunities).

*   The launch of **Ahrefs Connect**, a new OAuth-based system for third-party tool integrations using API v3.

*   Major updates across the platform, including a new **Possible 404 report** in Web Analytics and a **competitors map** in Keywords Explorer.

*   The deprecation of API v2 and legacy integrations, effective November 1st.

## Summary

The September 2025 update for Ahrefs is packed with 18 new features and improvements, headlined by significant expansions to Brand Radar and the introduction of a new integration framework.

### Brand Radar EnhancementsThe core of Brand Radar has been upgraded with two major features. **Entities** allow users to group multiple spelling variations or names for a brand or competitor into a single entity, ensuring all mentions are counted cohesively for a cleaner brand footprint view. **Topics** introduces a new report and overview widget that clusters search queries by parent topics from Keywords Explorer, helping users identify and prioritize new content opportunities. Additional Brand Radar updates include saved reports with access controls, a new filter builder for overview charts, dedicated filters for Cited Domains/Pages reports, Google Sheets export support, and the first endpoint of the new Brand Radar API for AI responses.

### Platform-Wide UpdatesMultiple core Ahrefs tools received substantial updates:

*   **Web Analytics**: Users can now compare funnels across dates, export tracked events directly, and access a new **Possible 404 report** that flags pages receiving visits but likely returning 404 errors, often due to AI chatbot hallucinations.

*   **Site Explorer**: A new interactive history chart for SERP features tracks owned and total feature positions over time. The Outgoing Links report now includes a status code filter to find broken links, replacing the old Broken Links report.

*   **Keywords Explorer**: A new competitors map in Traffic Share reports visualizes who is winning keywords by traffic and value. A historical trend chart shows shifts in traffic distribution, and a new filter allows exclusion of keywords already in lists or projects.

*   **Batch Analysis 2.0**: Now shows backlink counts, referring domains, and outlink data, fully replaces the old version with support for 1,000 URLs, and is faster and more accurate. Corresponding fields have been added to the Batch Analysis API v3 endpoint.

*   **Rank Tracker**: User notes in Rank Tracker and GSC charts are now shared across the entire workspace.

### Ahrefs Connect & API ChangesThe update announces **Ahrefs Connect**, a new OAuth-based system for third-party integrations built on API v3. It allows other tools to pull Ahrefs data like competitive intelligence and site audits directly into their workflows. Extra API units have been added to Lite, Standard, and Advanced plans for Connect usage. The company confirms that **API v2 and all legacy integrations will stop working on November 1, 2025**, marking a complete transition to the new API v3 infrastructure.

## Context

This update is critical for SEO professionals, digital marketers, and agencies who rely on Ahrefs for competitive analysis, keyword research, and backlink tracking. The shift to API v3 and Ahrefs Connect reflects the industry's move towards more integrated, automated workflows, while features like the Possible 404 report directly address new challenges created by the proliferation of AI-generated content and chatbots. The deprecation of older APIs necessitates immediate action for users with custom integrations.
