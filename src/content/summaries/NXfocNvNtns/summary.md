## Key Takeaways

This video clarifies the differences between Claude's three new remote interaction methods and provides usage recommendations. • **Claude Channels** enables interaction via messaging apps (Discord/Telegram) with **Claude Code** but has limited edit acceptance capabilities without 'run dangerously' mode. • **Dispatch** works exclusively with **Claude Co-work** desktop app for remote task execution and is not a Claude Code feature. • **Remote Control** via the official Claude phone app provides live terminal streaming with Claude Code and is recommended for most users seeking phone access.

## Summary

Anthropic recently introduced two new ways to interact with Claude remotely, bringing the total to three distinct methods: Claude Channels, Dispatch, and Remote Control. Each serves different purposes and works with different Claude products.

### Claude Channels

This is the newest method that allows users to talk to **Claude Code** through messaging platforms like Discord or Telegram. The channel-based approach is described as more "dev-focused" and "hacky," potentially offering additional functionalities for tinkerers. However, it has significant limitations: users cannot accept certain edits or changes to their machine unless they enable "run dangerously" mode, which creates security concerns.

### Dispatch

This method works exclusively with **Claude Co-work**, the desktop application, and is not related to Claude Code. Dispatch allows users to remotely instruct Claude Co-work to execute tasks autonomously, with the system reporting back when completed. This is essentially a remote command interface for the Co-work desktop app rather than a coding assistant feature.

### Remote Control

Available for about a month, this method works through the official Claude app on mobile devices and connects to **Claude Code**. It functions similarly to live streaming what's happening in your terminal, providing real-time interaction and the ability to accept edits and changes to your machine.

### Recommendation

The presenter clearly recommends **Remote Control** for "99% of people" who want to interact with Claude Code from their phones, citing its superior functionality and edit acceptance capabilities compared to Channels. The main distinction between Channels and Remote Control boils down to the apps being used (messaging platforms vs. official app) and the level of control they provide over system changes.
