## Summary

Anthropic has rolled out a significant update to Claude Code, its terminal-based AI coding agent, packed with features that enhance its autonomy, reliability, and usability.

### Skill Creator & Automated Testing

The standout feature is the enhanced **Skill Creator**, which now includes **built-in eval generation** with benchmarking and multi-agent testing. This allows skill creators to verify their skills work correctly, detect regressions, and automatically improve skill descriptions as models evolve. For example, a PDF parsing skill can be tested against forms with missing fields, and the system can automatically generate a fix. This brings software-style testing and iteration to AI agent skills without requiring users to write the code themselves.

### Enhanced Reasoning & New Modes

The update reintroduces **Ultra Think mode**, which extends the reasoning budget for models like Opus and Sonnet. This allows Claude Code to think longer before answering, leading to better performance on complex coding tasks, debugging, and multi-step agent workflows. For instance, when prompted to build a full-stack calendar app with React and TypeScript, Ultra Think mode enables more thorough planning and component creation.

New accessibility features include **Remote Control**, now available to all paid users, which lets you control and run Claude Code from your phone. Additionally, a **Voice Mode** is rolling out, allowing users to hold the spacebar and speak commands, which are then transcribed and executed—perfect for quickly initiating tasks like creating a SaaS landing page.

### Built-in Tools & Integrations

Several built-in tools improve the developer experience. A new **Claude API skill** automatically detects your programming language and loads the relevant API docs and references directly into your session. The **`/simplify` command** reviews changed code for reuse opportunities, code quality, naming, and efficiency, then applies fixes. It can spawn parallel agents using Haiku 4.5 for fast analysis.

Furthermore, a new **Slack plugin** integrates directly with Claude Code, allowing the AI to search messages, send updates, and create documents, bringing full team context into the development workflow.

These updates solidify Claude Code's position as a powerful, autonomous development partner capable of handling more complex and reliable software development tasks.

## Context

This update is significant as it represents a major leap in making AI coding agents more reliable, testable, and integrated into real-world development workflows. The focus on automated skill testing addresses a critical challenge in AI-assisted development: ensuring that agent capabilities remain consistent and improve over time. Developers and engineering teams looking to incorporate AI agents into their daily work should pay attention, as these features move Claude Code from a helpful assistant toward a more autonomous and dependable collaborator.
