## Key Takeaways

Anthropic's latest Claude Code update enables remote terminal control from mobile devices. • The new **`/remote` command** (version 2.1.52+) creates a cloud terminal session accessible from your phone. • This allows **mobile interaction with AI agents** like Claudia that can execute tasks using your computer's context and tools. • The feature makes Claude Code **10x more useful** by enabling execution on always-on devices like a Mac Mini.

## Summary

Anthropic has released a significant new feature for Claude Code that transforms how developers interact with their AI coding environment. The update introduces a **`/remote` command** that creates a cloud-based terminal session accessible from mobile devices.

To use the feature, you need to update to **Claude Code version 2.1.52** or higher. Once updated, simply type `claude` in your project directory and use the `/remote` command. This establishes a connection that can be accessed from your mobile device through the Claude Code interface.

The presenter demonstrates this functionality using **Claudia**, his open-source AI agent built on top of Claude Code. From his phone, he can send commands like "Hey Claudia, check my email and tell me what's going to happen tomorrow" which then execute on his computer terminal. The agent maintains full access to the computer's context and tools, enabling complex task execution.

This remote control capability dramatically expands Claude Code's utility. The presenter notes it makes the tool "10 times more useful" because it enables execution on always-on devices like his Mac Mini. Rather than being tied to a specific workstation, developers can now interact with their AI coding assistant from anywhere using their phone.

The feature effectively turns Claude Code into a **mobile-first development environment** where AI agents can perform tasks remotely while maintaining full access to local development resources. This represents a significant step toward more flexible, location-independent development workflows powered by AI assistants.
