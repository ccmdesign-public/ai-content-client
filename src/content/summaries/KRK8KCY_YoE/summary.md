## Key Takeaways

GPT-5.4 is a significant leap in AI capabilities with a focus on practical application and efficiency.

*   **Native computer use** enables the model to operate desktop UIs, achieving 75% on the OSWorld benchmark, a massive improvement from GPT-5.2's 47%.

*   **Major performance gains** include a 20% improvement over GPT-5.2 at similar cost, an 83% score on the GDP knowledge work benchmark, and a 1 million token context window.

*   **New efficiency features** like **tool search** (reducing token usage by ~47%) and interruptible **chain-of-thought** reasoning make the model faster and more cost-effective for coding and agentic tasks.

## Summary

OpenAI's GPT-5.4 represents a major step forward in creating AI models that can interact with the world like a human. Its most groundbreaking feature is **native computer use**, allowing it to click through and navigate desktop interfaces. This is quantified by a 75% score on the OSWorld benchmark, a nearly 30-point leap from GPT-5.2, making it the first OpenAI model to surpass average human performance on this task.

Beyond computer control, the model shows substantial overall improvements. It boasts a **1 million token context window** (with experimental API support) and scores **83% on the GDP benchmark**, a key metric for professional knowledge work across 44 occupations. Compared to GPT-5.2, it delivers roughly a **20% performance increase** while being more **token-efficient**, though it is slightly more expensive per token.

### Coding and Agentic Performance

Coding capabilities have been integrated from the specialized GPT-5.3 Codex model into this general-purpose release. On benchmarks like SWE-bench Pro, it performs similarly to GPT-5.3 Codex (~58%), representing a large jump from GPT-5.2. A key finding is that performance on **medium reasoning effort** settings has improved the most, offering an optimal balance of speed and accuracy for code generation.

### New Efficiency and Control Features

The release introduces two critical features for developers. First, **tool search** dynamically loads tool definitions on-demand instead of upfront, cutting token usage by about 47% in agentic workflows. Second, users can now **interrupt the model's chain-of-thought** reasoning mid-process to provide new guidance, allowing for real-time steering without restarting tasks.

### Benchmarks and Positioning

On independent benchmarks, GPT-5.4 shows strong design capabilities, jumping 9 positions on the Design Arena benchmark. While it leads on its 'extra high' setting, it slightly lags behind competitors like **Gemini 3.1 Pro** on normal settings for some tasks. The rapid release cadence—skipping a general GPT-5.3—highlights the intense pace of advancement, with this model consolidating coding prowess and new computer-use skills into a single, more efficient package.

## Context

This release marks a pivotal shift towards AI models that can directly manipulate digital environments, moving beyond pure text generation. The native computer use capability is a foundational step towards creating general-purpose AI agents that can perform tasks on any software with a standard UI. For developers, knowledge workers, and businesses, GPT-5.4's improvements in efficiency, coding, and real-world interaction lower the barrier to building sophisticated AI-assisted workflows and autonomous agents. It reflects the broader industry trend of models converging on multi-modal, tool-using intelligence.
