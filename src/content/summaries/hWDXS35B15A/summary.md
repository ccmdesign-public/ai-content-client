## Summary

Claude Code's latest update represents a significant leap towards autonomous AI agents with the introduction of **computer use**, allowing Claude to directly control your computer by pointing, clicking, and navigating just like a human user. This capability enables it to open files, browse the web, and interact with developer tools automatically without any setup required.

The **dispatch feature** acts as a remote control for Claude, letting users assign tasks from anywhere

- whether from the desktop app or a phone

- and have Claude execute them in the background. This is complemented by the **schedule command** which enables creating recurring Claude-based jobs from the terminal, turning Claude into a persistent worker that runs even after closing your laptop.

For developers, the update introduces **DOM element selection** in the desktop app, allowing users to directly click on UI elements instead of describing them. Claude then receives the HTML tag, classes, key styles, surrounding DOM context, and even a cropped screenshot of the element. For React apps, it goes deeper by accessing the actual source file, component name, and props.

Other notable improvements include **loops that can now run for up to 7 days** (up from 3), **configurable effort levels** for controlling how long Claude thinks before responding, and significant **performance improvements** with a 65% reduction in time to first bite and 50% faster component loading due to a move from SSR to a static architecture using Vite and TanStack Router.

The update also includes an **improved initialization command** that interviews users to help set up Claude Code in repositories, and introduces **projects in co-work** with built-in memory across tasks. Safety features are built-in with active monitoring for suspicious behavior and requiring user permission before accessing new apps or taking sensitive actions.

## Context

This update represents a major shift in AI capabilities, moving beyond simple chat interfaces to fully autonomous agents that can interact with and control computer systems. It's particularly relevant for developers, automation specialists, and anyone looking to delegate repetitive computer tasks to AI. The computer control feature, while powerful, raises important safety considerations that Anthropic has addressed with permission-based access and monitoring systems.
