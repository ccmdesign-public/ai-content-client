## Key Takeaways

A workflow for quickly creating new designs by remixing elements from existing websites. • Use the **HTML to design** Chrome extension to copy live websites directly into Figma. • Use **Magic Path** AI to mix design elements (like color themes) from different imported sources. • Generate and open production-ready code in **Cursor** with a single command from the remixed design.

## Summary

The presenter, Lukas Margerie, introduces a specialized design-to-code workflow that begins by capturing live websites. Using the **HTML to design** Chrome extension, you can select a viewport (desktop or mobile) on a site like granola.ai and send it directly to a Figma plugin. The plugin imports the design with options for auto-layout and local styles, though some fonts may need to be downloaded and updated manually.

Once in Figma, you can import multiple design sources. The video shows examples including granola.ai, Stripe, ramp.com, and a portfolio template from the Figma community. The core "remix" step happens in **Magic Path**. You copy the link to a design selection (e.g., Command+L in Figma) and paste it into Magic Path alongside another design. By selecting both and using a text prompt (e.g., "implement the color theme of the granola hero section"), the AI generates a new, combined design.

The final step is generating code. From the remixed design in Magic Path, you click 'Code' to get a command. Pasting this command into a terminal opens the project directly in the **Cursor** code editor. Running the project (e.g., `yarn dev`) launches a local server, rendering the newly created design that blends elements from the original sources. This process enables rapid ideation by combining the visual language of different successful websites into a new, coded prototype.

## Context

This workflow matters for designers, developers, and startup founders looking to rapidly prototype new projects or landing pages by drawing inspiration from successful, existing designs. It connects to broader trends in AI-assisted design and development, where tools are streamlining the process from visual inspiration to functional code. It's particularly useful for those wanting to experiment with different design styles without starting from a blank canvas, accelerating the initial stages of product development.
