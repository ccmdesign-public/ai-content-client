## Key Takeaways

Ahrefs' October 2025 release focuses on AI integration and workflow improvements for SEO and content professionals.

## Summary

The video details 16 new features and updates in Ahrefs for October 2025. The headline update is the launch of the **remote Ahrefs MCP server**, now available for all Lite plans and above. This allows users to connect AI assistants like **ChatGPT, Claude, or Copilot** directly to Ahrefs. The hosted server requires no setup, enabling AI tools to pull and analyze fresh Ahrefs data without accessing the main interface, though usage is subject to plan-based row and API unit limits.

Significant enhancements have been made to **Brand Radar** for tracking brand mentions and AI citations. The September data update for chatbots is now live, with ChatGPT and Perplexity having 5 months of historical data and Gemini and Copilot having three. Entities now support URL tracking alongside name-based mentions, consolidating all citations under a single entity. A new **citations chart** on the overview page visualizes mentions matched by name variations and impressions/AI share of voice matched by name or URL. Usability upgrades include a chart added to the AI responses report, nested tables in the topics report for easier previews, and reports now save filters, feature an optional name field, and a duplicate report option.

**Web Analytics** has a restructured report navigation with all reports now accessible from the sidebar. A practical upgrade allows users to copy and paste multiple URLs into page filters for analyzing several pages simultaneously.

In **Site Explorer**, the organic keywords report now shows which keywords belong to each tracked entity (brand, product, person) and allows filtering by them, helping identify visibility drivers and coverage gaps. The branded vs. non-branded traffic graph has been refined to split branded traffic into "your brand" and "other brands" for clearer competitor tracking. Filter presets are now shared between the Top Pages and Organic Keywords reports, and URL Rating is now available in Top Pages for sorting, filtering, and API access to gauge page strength.

**GSC Reports** now display SERPs (Search Engine Results Pages) directly within the GSC Keywords report, showing your ranking alongside competitors without needing to switch tabs. The system automatically shows the most relevant SERP based on the selected date range and clicks, using desktop results for consistency.

**Social Media Manager** introduces a major quality-of-life update: **multi-channel publishing**. Users can draft a post once, preview it per social platform, and then schedule or publish it to all selected channels simultaneously.

The final updates are for the **Ahrefs API**. A new spam filter has been added across all backlink-related endpoints and is also accessible via the MCP server, allowing AI assistants to pull clean backlink data. The **Page Explorer endpoint** is now live, providing access to hundreds of crawl fields and metrics for data comparison. It can be paired with the Site Audit issues endpoint to pull URLs affected by specific issues. For Rank Tracker, a new endpoint provides competitor metrics including share of voice, traffic, and position stats, with keyword-level filters promised for a future release.

## Context

This update is crucial for SEO professionals, content marketers, and digital agencies using Ahrefs for competitive analysis and strategy. It reflects the growing trend of integrating AI assistants directly into specialized SaaS tools to streamline data analysis and decision-making. The enhancements across Brand Radar, Site Explorer, and the API focus on providing deeper, more actionable insights into brand visibility, competitor performance, and content optimization in an increasingly AI-driven search landscape.
