- **Prioritize CLI tools over MCPs** for AI coding agents because they are purpose-built for terminal use and avoid the overhead of MCP servers.

## Summary

The video begins by addressing the overwhelming number of plugins available for Claude Code and aims to curate a list of the 10 most valuable ones based on the creator's personal use. A key theme is the preference for CLI tools over MCP servers, as CLIs are better suited for terminal-based AI agents.

### Supabase for Databases & Auth

The first tool discussed is the **Supabase CLI**, presented as a superior alternative to its MCP server. It's highlighted for handling databases (including vector DBs for RAG) and authentication with a generous free tier. The installation is a simple command, and after adding the corresponding skill, Claude Code can manage Supabase resources using plain language.

### Core Productivity Plugins

**Anthropic's Skill Creator** is deemed the most powerful tool for creating, evaluating, and improving custom skills with data-driven insights. **GSD (Get Stuff Done)** is an orchestration framework that provides guardrails for spec-driven development and manages context rot by using fresh context windows and sub-agents for new project creation.

**NotebookLM-PI** connects NotebookLM to Claude Code, enabling research, analysis, and content creation (videos, infographics) via the terminal despite no official API. **Obsidian** integration is recommended for personal assistant contexts, organizing sprawling markdown files by simply pointing Claude Code at an Obsidian vault folder.

### Deployment & Automation

**Vercel CLI** simplifies deployment management with a great free tier and should be paired with its provided skill. **Playwright CLI** from Microsoft enhances browser automation for tasks like web scraping, UI testing, and form submissions, and includes a visual dashboard (`playwright show`) to monitor headless agents.

**GitHub CLI** allows full GitHub management (pushes, PRs) directly from Claude Code. Since Claude Code is already adept with Git, a dedicated skill may not be necessary.

### Specialized Tools

**Firecrawl CLI** is tailored for AI-optimized web scraping with commands for scraping, crawling, mapping, and searching, making it superior to Claude's built-in web search for tasks like competitor research. **Excalidraw Diagram Skill** allows for creating presentation-ready diagrams via natural language, saving significant manual effort.

The creator concludes by advising viewers to find a "sweet spot" with plugins, warning that increased capability does not automatically equal increased performance.
