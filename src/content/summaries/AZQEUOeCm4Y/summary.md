## Key Takeaways

This video demonstrates a no-code method for web scraping using Claude Co-work and Apify.

*   Connect **Claude Co-work** to **Apify** via its MCP connector using an API key to access a marketplace of pre-built scraping 'actors'.

*   You can execute complex, multi-step research tasks entirely through natural language prompts, such as finding businesses without websites or compiling lead lists.

*   This setup enables powerful automation workflows when combined with Claude's scheduled tasks, like daily competitor monitoring or automated lead generation and outreach.

## Summary

The video provides a step-by-step guide to setting up no-code web scraping by integrating Apify with Claude Co-work. The process begins by installing the Claude desktop app with a paid subscription, then navigating to Co-work's connectors menu to install the 'Apify' MCP server connector.

### Setup Process

You need to create a free Apify account to obtain an API key, which is then entered into the Claude connector settings. A critical, unmentioned step is restarting the Claude desktop app for the new connector to be recognized. Once enabled, you can instruct Claude to 'use Apify' for any scraping task.

### Practical Examples

The tutorial walks through three escalating examples:

*   **Basic:** Asking Claude to use Apify to find a YouTube channel's subscriber count.

*   **Intermediate:** Researching plumbers in Reading, UK, who lack a website on their Google Maps profile, demonstrating lead generation for a web design business.

*   **Advanced:** Finding VC companies in London that are hiring, identifying the head of marketing for each, and compiling the data into a spreadsheet.

### Automation Potential

The real power is unlocked by combining this with **Claude Co-work's scheduled tasks**. The host shares use cases from his mastermind group:

*   Automatically checking competitors' new ads daily.

*   Monitoring LinkedIn for new job postings, researching the company, finding the right contact, and drafting a personalized outreach email.

*   Other applications include monitoring competitor pricing, reviews, backlink profiles, tracking newly funded companies, and aggregating customer sentiment from forums.

## Context

Web scraping is a powerful tool for market research, lead generation, and competitive intelligence, but it traditionally requires programming knowledge to bypass anti-scraping measures and parse website data. This tutorial democratizes access by leveraging AI (Claude) and a pre-built tool ecosystem (Apify). It matters for entrepreneurs, marketers, sales professionals, and business owners who need to gather market data efficiently without a technical team. It connects to the broader trend of AI agents automating complex, multi-step workflows that were previously manual or required significant development effort.
