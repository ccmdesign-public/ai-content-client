* The **OpenSpace** repository automatically tracks and optimizes agent skills, reducing token consumption by up to 46 percent on real-world tasks.

## Summary

The video explores five powerful open-source repositories released in March 2026 that significantly enhance the capabilities of Claude Code.

### Auto Research by Karpathy

The most prominent tool discussed is **Auto Research**, created by Andrej Karpathy.

This repository functions as an automated machine learning loop that allows Claude Code to run continuous, iterative tests to self-improve a codebase.

It requires objective, binary metrics to score the agent's performance, committing successful code changes while reverting failures via Git.

Shopify CEO Toby utilized this tool on a 0.8 billion parameter internal model, running 37 automated experiments over 8 hours to achieve a 19 percent efficiency gain.

### OpenSpace and Skill Optimization

Developed by the Hong Kong Data Intelligence Lab, **OpenSpace** is an MCP server designed for self-improving agent skills.

It monitors how an agent uses specific skills and categorizes them to either autofix, autoimprove, or autolearn.

By eliminating repetitive errors and optimizing successful actions, this tool reportedly reduces token usage by 46 percent on real-world professional tasks.

In one showcase, the system autonomously expanded its capabilities from six initial skills to 60 while building a complex monitoring dashboard.

### CLI Anything

The **CLI Anything** repository addresses the limitation of AI agents interacting with traditional software interfaces.

It operates an automated pipeline that analyzes open-source projects, writes tests, documents the code, and publishes it as a functional command-line interface tool.

This allows Claude Code to directly interact with complex software like Blender, OBS, and Zoom entirely through the terminal.

### Claude Peers for Multi-Agent Workflows

**Claude Peers** is a specialized tool that connects multiple independent Claude Code terminal sessions.

Using an MCP server and an SQLite database, it shares context and summaries across different agent instances.

This is particularly valuable for complex applications like video game development, where a single agent struggles to objectively evaluate its own work.

By connecting sessions, developers can create distinct planner, executor, and evaluator agents that communicate seamlessly.

### Google Workspace CLI

Finally, the **Google Workspace CLI** connects Claude Code directly to a user's entire Google Suite.

Built by Google developers, it transforms the agent into a highly capable personal assistant with access to Drive, Docs, and Gmail.

To mitigate security risks like prompt injections from external emails, the repository integrates Google's **Model Armor** to scan and neutralize malicious inputs before they reach the AI.
