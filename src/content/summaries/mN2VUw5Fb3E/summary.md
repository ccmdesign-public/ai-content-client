## Summary

The video provides a step-by-step guide to using Claude Code, an agentic AI coding assistant that operates in the terminal, without any cost. This is achieved by exploiting a key feature: **Ollama's Anthropic API compatibility**. This allows Claude Code to interface with open-source large language models hosted locally via Ollama, instead of requiring a paid connection to Anthropic's official models.

### **Core Setup Process**

The setup involves three main components:
1.  **Install Ollama**: Download and install Ollama from its official website, which will manage the local LLMs.
2.  **Install Claude Code**: Install the Claude Code agent via terminal commands appropriate for your operating system.
3.  **Configure the Connection**: Use specific environment variable commands in your terminal to point Claude Code to your local Ollama server, effectively tricking it into using the local models.

### **Model Selection and Performance**

Not all local models are equal for this task. The creator recommends using models with at least a **32K token context window** for effective development workflows. Specific recommendations include:

*   **Qwen 3.5 7B/27B**: Highlighted as a top performer, with the 7B version being more accessible and the 27B offering stronger intelligence if your hardware supports it.

*   **GLM 4**: Another viable open-source option mentioned for use with this setup.
The creator demonstrates the setup by installing the **Qwen 3.5 7B model** (a 17GB download) and shows it successfully generating a complete landing page through Claude Code's agentic process.

### **Important Considerations**

A critical prerequisite is ensuring your system, particularly your **GPU**, has sufficient **VRAM** to run the chosen model smoothly while leaving room for large code context windows. The presenter uses an **NVIDIA GeForce RTX 4090** as an example. He mentions using online tools to check which models are compatible with specific GPUs (like an RTX 3090) based on VRAM, required features, and desired context size.
To start a Claude Code session with the local model, you must launch it from a new terminal with a specific command that includes the model name (e.g., `claude --model qwen3.5`), rather than just using the standard `claude` command.
