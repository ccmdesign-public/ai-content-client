## Key Takeaways

Claude Code has introduced major features that directly challenge OpenClaw's functionality. • **Remote Control** enables developers to manage local coding sessions from their phones while away from their desks. • **Scheduled Tasks** in Co-Work automate recurring workflows like daily briefings and weekly updates without external automation layers. • These features are available to paid tier users and demonstrate Enthropic's strategy of integrating OpenClaw-like capabilities directly into their core products.

## Summary

Enthropic has responded to the competitive threat from OpenClaw by shipping two major new features in Claude Code that directly replicate and enhance OpenClaw's core functionality.

### Remote Control Feature

The new **Remote Control** feature allows developers to start a coding session in their terminal and continue managing it remotely through the Claude app on their phone. This means developers can step away from their computers while their local development environment stays connected and MCP servers keep running. The feature is currently in research preview for Mac users with broader rollout planned for coming weeks, accessible via the `/remote-control` command.

### Scheduled Tasks in Co-Work

The second major update introduces **Scheduled Tasks** within Co-Work, enabling Claude to automatically run recurring workflows like daily morning briefings, weekly spreadsheet updates, team presentations, and ongoing operational automations. There's no external automation layer required, and the feature includes a new customized tab in the Co-Work sidebar that acts as a central hub for managing plugins, skills, connections, and integrations.

The video demonstrates a practical example where Claude is scheduled to scan AI news sources every morning at 8:00 AM using the Sonnet 4.6 model, which then generates a comprehensive daily AI recap including top stories, trending GitHub repositories, and YouTube video ideas. This automation runs autonomously while the user sleeps and can be controlled from mobile devices.

### Competitive Landscape

The presenter argues that while OpenClaw gained viral attention for showcasing automated computer workflows, similar systems like Agent Zero and Lemon existed years earlier. The real threat to OpenClaw comes from major AI labs like Enthropic integrating these capabilities directly into their core products, making standalone frameworks less necessary. This trend is also visible with Notion introducing custom agents and Perplexity unifying AI capabilities into their system.

Ultimately, the combination of remote session management and powerful scheduled automation positions Claude Code as a compelling alternative to OpenClaw, especially as these features become more widely available to Claude users.

## Context

This update matters because it represents a strategic shift in the AI assistant landscape where major labs are integrating automation capabilities directly into their products rather than relying on third-party frameworks. Developers and teams using AI for workflow automation should pay attention as these integrated solutions offer simpler setups and potentially better reliability. The trend reflects broader industry movement toward autonomous AI agents that can work continuously without human supervision, with companies like Notion and Perplexity also introducing similar agent capabilities.
