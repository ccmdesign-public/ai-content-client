*   **Skills are reusable playbooks** that package specific AI workflows, eliminating the need to start from scratch each time (e.g., a 'Go Places' skill queries the Google Places API).

*   **You can easily combine skills** in a single chat to build complex outputs, such as generating restaurant research and then turning it into a dark-mode HTML presentation.

## Summary

The video introduces **HappyCapy**, a platform for building and using **AI agent skills**. These skills act as reusable playbooks for specific tasks, moving beyond one-off prompts. The presenter explores the platform's skills store, which features community-contributed tools for tasks like querying APIs or creating presentations.

### Using and Combining Skills

The demonstration begins by installing and using the **'Go Places' skill**, which queries the Google Places API to find highly-rated restaurants in a given location (e.g., Wynwood, Miami). The results are then passed to the **'Frontend Slides' skill**, which transforms the raw research into a polished, dark-mode HTML presentation complete with clickable links to restaurant websites and Google Maps.

### Building a Custom Skill

The core workflow shows how to **package this entire multi-step process into a new, custom skill**. Using the built-in **'HappyCapy Skill Creator'**, the presenter instructs the AI to analyze the chat history and generate a new skill called **'Offsite Venue Finder'**. This skill automatically creates the necessary directory structure and documentation.

Once created, the new skill can be run independently. The presenter tests it by asking for top restaurants in Soho, New York City, and the skill autonomously repeats the entire workflow—researching, finding websites, and generating a presentation with the same design template.

### Enhancing and Iterating

The video also shows how to **iteratively improve a skill** by adding new capabilities. After generating the initial presentation, the presenter installs an image generation skill (**'Nana Banana Pro'**) and uses it to add a custom header image to the presentation. This enhanced output can then be fed back into the Skill Creator to update the original 'Offsite Venue Finder' skill, making the new feature part of its standard output.
